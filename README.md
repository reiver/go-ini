# go-ini

Package **ini** provides tools for working with the INI data format, for the Go programming language.


## INI Format

The INI data format looks like this:
```ini
; This is a comment.

# This is also a comment.

[this is a section]
key=value

apple=one
banana=two
cherry=three


[person]
name=Joe Blow
organization=Acme Inc.

```
